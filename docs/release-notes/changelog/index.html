SHKeeper Changelog

This changelog covers all official updates from SHKeeper.

November 20, 2025 — Bitcoin Support via Public Node & Reduced Server Requirements​

SHKeeper adds support for Bitcoin through public nodes and reduces server requirements for Bitcoin and other cryptocurrencies. shkeeper.io

November 7, 2025 — TRON Staking for Lower TRC‑20 Fees​

Introduced TRON Staking, including Energy delegation, Bandwidth management, and optional Super Representative voting to reduce TRC‑20 fees. shkeeper.io

September 8, 2025 — Ethereum Internal Transaction Detection​

Detects and processes Ethereum internal transactions (smart contract-based transfers) correctly for invoice status updates. shkeeper.io

June 30, 2025 — BNB Module Update​

Optimizes BNB integration performance and updates default BNB fee configuration. shkeeper.io

June 6, 2025 — XRP Reserve & Fee Reductions​

Reduced XRP reserve and deletion fees to lower operational costs. shkeeper.io

February 14, 2025 — Solana Network Support​

Added Solana network support (SOL, USDT, USDC, PYUSD). shkeeper.io

January 20, 2025 — Critical Bug Patch​

Fixed a critical bug affecting currency rate calculations; users were advised to update immediately. shkeeper.io

December 20, 2024 — Lightning Network Support​

Added Lightning Network support for fast, low-fee Bitcoin payments. shkeeper.io

December 19, 2024 — External Node Capability for Monero​

Ability to use an external Monero node to reduce server requirements. shkeeper.io

December 11, 2024 — Temporary Tron Fullnode Issue Resolved​

Temporary issue with the TRON fullnode resolved. shkeeper.io

December 3, 2024 — XRP Payment Handling Bug Fixed​

Critical bug affecting XRP payment handling resolved. shkeeper.io

October 4, 2024 — New Crypto Price Sources & Dashboard Enhancements​

Added multiple crypto price sources (KuCoin, Kraken, Coinbase) and enhanced the admin dashboard. shkeeper.io

June 25, 2024 — Avalanche (AVAX) Support​

Added Avalanche network support. shkeeper.io

June 21, 2024 — Polygon (MATIC) Support​

Added Polygon network support. shkeeper.io

May 31, 2024 — Several SHKeeper Instances at the Same Time​

Ability to run multiple SHKeeper instances on the same server using namespaces. shkeeper.io

May 9, 2024 — External Node Capability for BNB​

Support for using an external BNB node (BNB, BEP20 USDT, BEP20 USDC). shkeeper.io

April 23, 2024 — External Node Capability for Ethereum​

Option to use an external Ethereum node for ETH and ERC‑20 assets. shkeeper.io

January 12, 2024 — Ripple (XRP) Support Added​

Native support for Ripple payments added. shkeeper.io

September 19, 2023 — Binance Coin (BNB) & General Features Improvements​

Improvements for Binance Coin and general functionality. shkeeper.io

August 4, 2023 — Interface Improvements​

Introduced UI enhancements and wallet encryption updates. shkeeper.io

March 20, 2023 — Monero (XMR) Support Added​

Monero support added. shkeeper.io

February 14, 2023 — Private Key Download & Import Feature​

Added secure download and import of private keys. shkeeper.io

January 17, 2023 — Ethereum & ERC20 Support​

Support for ETH, USDT (ERC20), USDC (ERC20) added. shkeeper.io

November 6, 2022 — Multipayout API Introduced​

Multipayout API for multi-address payouts and dashboard added. shkeeper.io

October 21, 2022 — USDC (TRC‑20) Support​

Added USDC support on TRC‑20. shkeeper.io

September 3, 2022 — Tron (TRX) Support Added​

Native Tron support added. shkeeper.io

August 18, 2022 — USDT (TRC‑20) Support​

Support for USDT on TRC‑20 added. shkeeper.io

July 29, 2022 — WHMCS Module Released​

SHKeeper module released for WHMCS integration. shkeeper.io

May 16, 2022 — Advanced Improvements & Open Source Release​

Advanced improvements and open-source release. shkeeper.io

January 4, 2022 — First Public Release​

First SHKeeper public release with basic crypto payment capabilities. shkeeper.io
