## SHKeeper Demo

SHKeeper provides an official **demo version** so you can try the system without installing it yourself. The demo runs in a test environment and lets you explore the core options and features of the payment processor.
- **Demo URL:** https://demo.shkeeper.io  
- **Login:** `admin`  
- **Password:** `admin`
You can also watch the official video guide on YouTube that walks through the SHKeeper demo and main features: [SHKeeper demo walkthrough](https://www.youtube.com/watch?v=yYK_JAm1_hg).
The demo gives you practical experience using SHKeeper’s interface and configuration options before deploying your own instance.
